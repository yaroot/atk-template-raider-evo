# Paper template for ATK (Free)Raider EVO

This is the paper template for ATK (Free)Raider EVO, works with all raider evo bindings plus the R16A toe plate.
The one on ATK's website doesn't have marking for the toe plate, which is IMHO useless.

## Notes

- print with 100% scaling, should work with both A4 and US letter size
- measure after printing
- measure again

## Notes for first timer

- do yourself a favor by using threaded inserts on skis (just normal ones plus some epoxy will do)
- practice on some 2x4s
- practice on old skis
- if you're really paranoid install with a different offset first (if using theaded inserts)
- BSLs are not the same across brands, measure and try it on 2x4s
